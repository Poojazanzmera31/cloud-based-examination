============================================================
  TROUBLESHOOTING
============================================================

Problem: "Login failed" error
Solution: 
  - Make sure backend is running (http://localhost:5001)
  - Check browser console for errors (F12)
  - Verify MongoDB is connected

Problem: Page is blank/white
Solution:
  - Open browser console (F12) to see errors
  - Check if frontend is running
  - Try refreshing the page (Cmd+R)

Problem: Redirected to wrong page
Solution:
  - Clear localStorage in browser console:
    localStorage.clear()
  - Refresh the page
  - Login again

Problem: Cannot access /admin/dashboard
Solution:
  - You must be logged in as admin
  - If logged in as student/faculty, you'll be redirected
  - Logout and login with admin credentials
