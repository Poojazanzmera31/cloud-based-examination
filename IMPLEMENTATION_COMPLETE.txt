============================================================
  API ENDPOINTS IMPLEMENTED
============================================================

AUTH:
  POST   /api/auth/register     - Register new user
  POST   /api/auth/login        - Login (email or username)

USERS (Admin only):
  GET    /api/users             - Get all users
  GET    /api/users/:id         - Get user by ID
  POST   /api/users             - Create user
  PUT    /api/users/:id         - Update user
  DELETE /api/users/:id         - Delete user

EXAMS:
  GET    /api/exams             - Get all exams (with filters)
  GET    /api/exams/:id         - Get exam by ID
  POST   /api/exams             - Create exam (Faculty)
  PUT    /api/exams/:id         - Update exam (Faculty)
  DELETE /api/exams/:id         - Delete exam (Faculty/Admin)
  GET    /api/exams/my          - Get faculty's exams

RESULTS:
  POST   /api/results/submit    - Submit exam (Student)
  GET    /api/results           - Get student's results
  GET    /api/results/faculty   - Get faculty's exam results
  GET    /api/results/exam/:id  - Get specific exam results
  GET    /api/results/:id       - Get result by ID
